# 📝 Easy Puzzle Upload Guide

## No JSON Required! Simple Form Fields

---

## Strands Puzzle

### 1. Theme
Enter a theme that connects all the words.

**Example:**
```
Literary Genres
```

### 2. Letter Grid
Enter letters separated by spaces, one row per line.

**Example:**
```
P O E T R Y
N O V E L S
D R A M A S
```

### 3. Hidden Words
All findable words separated by commas.

**Example:**
```
POETRY, NOVEL, DRAMA, PROSE, VERSE
```

---

## Spelling Bee

### 1. Center Letter
Just one letter (the required letter).

**Example:**
```
O
```

### 2. Outer Letters
6 letters separated by commas.

**Example:**
```
B, K, S, R, E, A
```

### 3. Valid Words
All valid words separated by commas.

**Example:**
```
BOOK, BOOKS, BORE, BORES, BORED, BROKE, BROKER, SORE, ROSE, SOAR, ROBE, ROBES, SAKE, SOAK
```

## ✅ Format Rules

✅ One item per line for grid
✅ Commas separate letters and words for spelling bee
✅ Spaces separate letters in strands grid

❌ Don't use periods or special characters in strands grid
❌ Don't forget commas between words

---

## 🚀 Quick Steps

1. Go to **Admin Dashboard** (`/admin`)
2. Click **"Upload Games"** tab
3. Select game type (Strands or Spell Bee)
4. Enter today's date
5. Fill in the fields (copy examples above)
6. Click **"Upload Puzzle"**

Done! ✨

---

## 💡 Tips

- **Strands**: Make sure all words can be found by connecting adjacent letters
- **Spelling Bee**: All words must contain the center letter
- **Both**: Test your puzzle before uploading to ensure it's solvable
- **Date**: Use today's date for immediate availability

---

## ❓ Need Help?

If you see an error:
1. Check your format matches the examples exactly
2. Make sure there are no extra spaces or line breaks
3. Verify all required fields are filled
4. Try copying an example and modifying it

The system will show an error message if something is wrong!
